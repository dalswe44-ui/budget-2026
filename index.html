📧 Gmail 공과금 자동 불러오기

Google 계정을 연결하면 공과금·관리비 메일을 자동으로 읽어서 달력에 추가할 수 있어요.

⚠️ Gmail 연동을 사용하려면 Google Cloud Console에서 OAuth 앱을 등록해야 해요.
설정 방법:
1. console.cloud.google.com 접속
2. 새 프로젝트 생성 → Gmail API 활성화
3. OAuth 2.0 클라이언트 ID 생성 (웹 애플리케이션)
4. 아래 CLIENT_ID 변수에 발급받은 ID 입력
5. 승인된 리디렉션 URI에 이 페이지 URL 추가

📌 현재는 데모 모드로 샘플 공과금 데이터를 확인할 수 있어요.
